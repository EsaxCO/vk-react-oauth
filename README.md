# Run the project
<p>In .env file set your personal oauth_id and oauth_secret from vk dev application</p>

### `NPM`

```
npm install

npm run start:build
```

### `YARN`

```
yarn install

yarn start:build
```

<b>PS:</b> After the first building you can simply just use the command `npm run start:server` or `yarn start:server`
