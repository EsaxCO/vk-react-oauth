# Run the project
<p>In .env file set your personal oauth_id and oauth_secret from vk dev application</p>

### NPM

##### `SERVER`
```
npm install
```

##### `CLIENT`

```
cd client

npm install

npm run build
```

#### `Run the server`

```
npm run start
```


### YARN

##### `SERVER`
```
yarn install
```

##### `CLIENT`

```
cd client

yarn install

yarn build
```

#### `Run the server`

```
yarn start
```
